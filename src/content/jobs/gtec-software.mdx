En Gtec Software participe en varios proyectos para diferentes clientes que la empresa tenia.

#### Guardtec
    - Desarrolle dos aplicaciones con Flutter para la gestión de accesos a Fraccionamientos o Unidades habitacionales. La primer aplicación sería utilizada por los habitantes para poder acceder por medio de un QR que sería leído en la entrada y proporcionado por la aplicación, se podía dar acceso a visitantes, familiares, empleados o servicios. Dependiendo del acceso se llenaba un formulario con información detallada para una mejor administración y seguridad ya que cada acceso tenia que ser leído de entrada y salida con el fin de evitar algún percanse dentro de la unidad. La segunda aplicación era usada en la caseta en donde se registraba el acceso de las personas tomando una foto del visitante, foto de la identificación y en su caso las placas del auto, ya que se pedia la identificación también se registraba el número de cajón donde se guardaría para que en la salida fuera más fácil identificar a donde se guardo.
    - Tecnologías utilizadas: Flutter, API Rest, Dart.
#### Asistencia Víal - GNP Seguros
    - Fui parte del desarrollo de un módulo que se encargaba de reportar siniestros a los proveedores de la aseguradora, tomando foto e información del siniestro, se desarrollo el mapa de seguimiento para cada punto del proceso, desde la asistencia de la grúa hasta la visualización en tiempo real del estatus.
    - Tecnologías utilizadas: Flutter, Firebase, Dart.
#### Equipo dedicado - GNP Seguros
    - Fui parte de un equipo de desarrollo dedicado que se encargaba de ir resolviendo nuevos issues o realizando nuevos features a la aplicación de Soy Cliente con el fin de atender más tareas y mejorar la aplicación en un limite de tiempo.
    - Tecnologías utilizadas: Flutter, Angular, React, Firebase, Javascript, Dart.
#### Portal GNP Seguros

    - Tecnologías utilizadas: Angular, React, Javascript, API Rest, Firebase.
#### Regimen Fiscal - GNP Seguros
    - En la aplicación de Soy Cliente desarrolle un nuevo módulo donde se subía un archivo PDF de la constancia de regimen fiscal y esta se leía por medio de una API y con el resultado se autocompletaban todos los campos del formulario para realizar el registro de esta constancia.
    - Tecnologías utilizadas: Flutter, API Rest, Firebase.
